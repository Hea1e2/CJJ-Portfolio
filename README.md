# 成佳俊的个人作品集

> 一个展示 AI 内容运营与 AIGC 技术实践的在线作品集网站

## 📖 项目简介

这是一个现代化的个人作品集网站，用于展示我在 AI 领域的项目经验和 AIGC 创作作品。网站采用纯前端技术栈，设计简洁优雅，注重用户体验。

**核心理念**：为虚拟的世界，付出真实的情感

## ✨ 主要功能

### 1. 个人 AI 项目展示
- **项目分类**：支持按产品设计、技术实现、社会影响、硬件集成等分类筛选
- **详情展示**：
  - 对于有在线链接的项目：提供"在线体验"按钮直接访问
  - 对于有展示资源的项目：提供"查看详情"按钮查看图片/视频
  - 支持图片/视频轮播，完整展示项目细节
- **真实描述**：所有项目描述基于实际情况，不夸大不编造

### 2. AIGC 作品视频墙
- 瀑布流布局展示 AIGC 视频作品
- 鼠标悬停预览效果
- 点击弹窗全屏观看

### 3. 响应式设计
- 适配桌面端、平板和移动端
- 流畅的动画效果和交互体验

## �️ 技术栈

## 🚀 使用方法

### 本地运行

1. **克隆或下载项目**
   ```bash
   # 直接下载或复制项目文件到本地
   ```

2. **打开项目**
   - 直接双击 `index.html` 文件
   - 或使用本地服务器（推荐）：
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Node.js (需要安装 http-server)
     npx http-server
     ```

3. **访问网站**
   - 直接打开：在浏览器中打开 `index.html`
   - 本地服务器：访问 `http://localhost:8000`

### 添加新项目

1. 在 `AI项目/` 目录下创建项目文件夹
2. 将项目的图片/视频资源放入文件夹
3. 在 `js/data.js` 中添加项目配置：

```javascript
{
    id: 10,
    title: "项目名称",
    desc: "项目描述",
    category: "tech", // product/tech/social/hardware
    tags: ["标签1", "标签2", "标签3"],
    mediaType: "video", // 或 "image"
    src: "AI项目/项目文件夹/封面图片或视频.mp4",
    link: "https://项目在线链接.com", // 可选
    gallery: [ // 详情展示的资源列表
        { type: 'image', src: 'AI项目/项目文件夹/图片1.jpg' },
        { type: 'video', src: 'AI项目/项目文件夹/视频1.mp4' }
    ]
}
```

### 添加 AIGC 视频

1. 将视频文件放入 `AIGC/` 目录
2. 在 `js/data.js` 的 `aigcVideos` 数组中添加文件名：

```javascript
const aigcVideos = [
    "1.mp4", "2.mp4", ..., "新视频.mp4"
];
```
